üè¶ Predicting Customer Churn to Improve Retention Strategies

Overview Link to heading

  • Business Problem: Customer churn is a significant challenge for many industries including banking. The project aimed to predict churn using machine learning, identify its key drivers, and provide actionable insights to inform retention strategies
  • Dataset: 160,000+ bank product records from Kaggle.

Solution Link to heading

  • Data Preparation: Cleaning, scaling, and feature engineering to enhance prediction accuracy.
  • Model Training: Leveraging CatBoost with cross-validation to handle class imbalance and optimize performance.
  • Model Interpretation: Using SHAP (SHapley Additive Explanations) to understand feature importance, and derive actionable insights.

Approach Link to heading

  1. Data Preprocessing:

    • Handled missing values and outliers
    • Engineered features using WoE and IV
    • Created fine and coarse class bins for continuous variables

  2. Modeling:

    • Built a logistic regression model for Probability of Default (PD)
    • Used logistic and linear regression for Loss Given Default (LGD)
    • Used linear regression for Exposure as Default (EAD)

  3. Evaluation:

    • Evaluated PD model performance with AUC-ROC (0.69) and Gini Coefficient (0.38)
    • Assessed LGD model accuracy with calculating correlation between actual and predicted recovery rate (0.38)
    • Assessed EAD model accuracy with calculating correlation between actual and predicted CCF (0.53)

  • Hypothesis testing (Two-Proportion Z-test)
  • Diagnostic analyses
  • Social Media strategies

üß∑ TikTok In-app Safety Survey, 2021-2022

The Problem:
The TikTok Safety Product team wanted to understand how users felt about the safety of TikTok and what they can do to improve their perception. There were nine key safety areas back in 2021, like misinformation, hate speech, and minor safety, they needed user insights to guide their decisions on areas to prioritize.

My Actions: Me and my user research colleagues began the project, curated the questionnaires with cross-functional and market efforts, and sent out in-app survey targeting 1% randomly selected TikTok active users in specific markets. Results: By analyzing the feedback, we identified the key issue areas for users in several key markets and where users believed that TikTok was underperforming. We presented the report to relevant teams.

This research shaped the Safety Product team’s strategy, leading to the development and optimization of in-app features like Family Pairing and Anti-Addiction Reminders. It also guided the Global PR team’s 2022 roadmap, influencing plans to invest more in PR and marketing to raise awareness of these features.
