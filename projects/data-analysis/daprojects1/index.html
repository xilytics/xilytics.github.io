üè¶ Credit risk modelling with machine learning models in Python

Overview:

  • Business Problem: How can banks predict loan default risks, decide whether to grant loan to individuals or not, and potential losses?
  • Dataset: Analyzed 80,000+ US loan records from 2007-2014 from Kaggle, including features like income, employment, home ownership,loan purpose, address and so on.

Key Techniques and Tools Link to heading

  • Techniques: Weight of Evidence (WoE), Information Value (IV), Fine/Coarse Classing, Logistic and Linear Regression, Population Stability Assessment
  • Evaluation Metrics: AUC-ROC, Gini Coefficient, Kolmogorov-Smirnov (KS)
  • Tools: Python (scikit-learn, Pandas, Matplotlib, Seaborn)

Approach Link to heading

  1. Data Preprocessing:

    • Handled missing values and outliers
    • Engineered features using WoE and IV
    • Created fine and coarse class bins for continuous variables

  2. Modeling:

    • Built a logistic regression model for Probability of Default (PD)
    • Used logistic and linear regression for Loss Given Default (LGD)
    • Used linear regression for Exposure as Default (EAD)

  3. Evaluation:

    • Evaluated PD model performance with AUC-ROC (0.69) and Gini Coefficient (0.38)
    • Assessed LGD model accuracy with calculating correlation between actual and predicted recovery rate (0.38)
    • Assessed EAD model accuracy with calculating correlation between actual and predicted CCF (0.53)
